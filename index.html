OpenEXR Viewer

Inspect OpenEXR images in the browser with local file loading, raw pixel probes, channel views, ROI statistics, panoramas, and PNG export.

WebGL2 + exrs WASM. Local files stay in your browser.

Open Web App

Features

Tools

  • Local files Open EXR files from disk, drag and drop, or load sample scenes.
  • Raw probe Inspect source pixel values before exposure, gamma, or colormap transforms.
  • Channels Browse RGB, alpha, AOV, spectral, Stokes, and grouped channel views.
  • ROI stats Measure regions with bounds, valid samples, min, mean, and max.
  • Metadata Review EXR header metadata, channel lists, and image windows.
  • Export Save PNG images, screenshots, batches, LUTs, and reproduction metadata.

Details

  • OpenEXR 2.x scanline images
  • Half, float, and uint channels
  • WebGL2 renderer
  • exrs WASM decoder
  • Static-hosting ready
  • No uploads for local files
